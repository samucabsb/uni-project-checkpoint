# Melhorias da V1.1

## UX

- Cadastro retorna para login.
- Toast global de sucesso/erro.
- Loading em botões críticos.
- Feedback visível após avaliação.
- Estados vazios com CTA.
- Modais/confirm actions preparados via componente.

## UI

- Visual escuro, minimalista e social.
- Cards de jogos com capa, nota e metadados.
- Home pública com seções de valor.
- Perfil com favoritos, avaliações e listas.
- Página de jogo organizada em blocos.

## Produto

- Biblioteca pessoal.
- Status do jogo.
- Top 4 favoritos.
- Feed social.
- Admin dashboard.
